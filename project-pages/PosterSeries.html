Poster Series

A cohesive set of posters designed for a fictitious typography lecture series hosted by The New School
Project Details
This series of posters, pictured above and below, feature a number of type designers and promote a fictional lecture series hosted by The New School. I was given names, copy, and dates ahead of time. This project was done to focus on typographic hierarchy and font pairings and uses the typeface Oswald with minor design adjustments to characters to make them aesthetically work in this layout. This was done using Adobe Illustrator and Google Font.
