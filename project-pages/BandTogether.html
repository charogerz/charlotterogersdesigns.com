Band Together

An online multiplayer game I coded using the p5.party library, a developing library for p5.js multiplayer games
Project Details
I took a class the last semester of my graduate school experience to learn about the p5.party library and strengthen my creative JavaScript skills. My team members and I worked on this multiplayer puzzle game that mimics an in-person escape room, with clues, interactive elements, and a timer. I was the technical lead and did most of the coding for this project.
You can find the GitHub project page HERE and play the game for yourself HERE.
